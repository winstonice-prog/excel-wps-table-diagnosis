# Contributing

Thanks for taking a look at this project.

## What Helps Most

The most useful contributions are practical ones:

- clearer spreadsheet use cases
- better Excel and WPS compatibility notes
- examples based on real office tables
- simpler and more maintainable formula suggestions
- corrections to workflows that are too fragile in daily use

## Suggested Contribution Style

- keep examples small and realistic
- prefer plain language over product wording
- explain why a workflow is safer, not just why it is possible
- avoid adding features that make the project too broad

## Before Opening A PR

- check whether the change fits the current scope
- keep README and examples consistent
- prefer focused patches over large unrelated changes

## Scope Reminder

This repository is centered on:

- table diagnosis
- cleanup workflows
- lookup and matching
- summarization guidance
- Excel and WPS compatibility

It is not currently centered on:

- visual formatting
- chart generation
- full automation or scripting across office apps
