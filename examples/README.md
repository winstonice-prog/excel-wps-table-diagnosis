# Examples

This folder contains simple examples that show the current direction of the project.

## Files

- `table-diagnosis.md`: inspect a messy table and suggest cleanup order
- `sheet-match.md`: match two sheets with compatibility-aware lookup guidance
- `text-cleanup.md`: clean a text column with simple, maintainable steps

## How To Read These Examples

Each example is intentionally lightweight:

- input shape
- user request
- expected response shape

They are meant to document the workflow and output style before the project grows into a larger implementation.
